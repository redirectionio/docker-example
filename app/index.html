Welcome in the redirection.io docker demo

Please setup some redirection rules in the manager. You can download this CSV rules file, and import it in your redirection.io project.

Once the according rules have been created, you can test the links below:

  • Example 301 redirection (by redirection.io)
  • Example 302 redirection (by redirection.io)
  • Example 404 response (by nginx/backend)
  • Example 404 response (by redirection.io)
  • Example 410 response (by redirection.io)

Please note that you may have to wait a bit before the rules are applied - it can take up to 10 minutes on a free plan.
